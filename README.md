# Пересечение двух окружностей на плоскости

Лабораторная работа №1 по курсу «Разработка ПС», **вариант 11**: для двух заданных окружностей определить, есть ли у них общая область положительной площади, и вычислить **площадь пересечения** соответствующих круговых дисков.

**Исполнитель:** Шунин Михаил Дмитриевич.

## Возможности

- Ввод параметров двух окружностей с клавиатуры или из текстового файла.
- Сохранение примера входного файла и запись результата в файл (путь задаёт пользователь).
- Настройка путей к файлам ввода и результата через меню.
- Модульные тесты (xUnit).

Язык интерфейса консоли — русский. Идентификаторы в коде — на английском (по требованиям методички).

## Требования

- [.NET 8 SDK](https://dotnet.microsoft.com/download) — для локальной сборки и тестов **или**
- [Docker](https://www.docker.com/) с поддержкой Compose — сборка и запуск без установки SDK.

## Запуск через Docker Compose (рекомендуется)

Из корня репозитория:

```bash
docker compose build
```

**Интерактивная программа** (меню):

```bash
docker compose run --rm app
```

Флаги `-i`/`-t` для `compose run` не обязательны: в `docker-compose.yml` для сервиса `app` заданы `stdin_open` и `tty`.

**Юнит-тесты:**

```bash
docker compose run --rm test
```

## Локальная сборка без Docker

```bash
dotnet restore CircleIntersection.sln
dotnet run --project src/CircleIntersection.App/CircleIntersection.App.csproj
```

Тесты:

```bash
dotnet test tests/CircleIntersection.Tests/CircleIntersection.Tests.csproj
```

## Формат входного файла

Две строки, в каждой три числа через пробел: `cx cy r` — координаты центра и радиус окружности. Пример:

```
0 0 2
3 0 2
```

Числа с дробной частью — в формате с точкой (инвариантная культура).
