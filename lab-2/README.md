# Lab-2

Лабораторная работа №2 реализует вариант 1 из списка повышенной сложности:
поиск подматрицы с максимальной суммой элементов в двумерном массиве вещественных чисел.

- Вариант по списку: **22**
- Вариант задания для ЛР2: **1**

Формат входного файла:

```text
rows columns
a11 a12 ... a1m
a21 a22 ... a2m
...
an1 an2 ... anm
```

Первая строка содержит число строк и столбцов матрицы.
Каждая следующая строка содержит элементы одной строки матрицы в формате `double`
с использованием точки как десятичного разделителя.

Рекомендуемый запуск из каталога `lab-2/`:

```bash
docker compose build
docker compose run --rm app
docker compose run --rm test
```

При запуске через `docker compose` контейнеру проброшены `/Users` и `/Volumes`,
поэтому можно указывать абсолютные пути macOS. При сохранении файлов программа
автоматически создаёт недостающие папки.
